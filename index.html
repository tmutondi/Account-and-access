üìò 5. Training Guide (Summary)

Account Creation Steps:

  1. Receive and verify the access request form.
  2. Confirm managerial approval and role alignment.
  3. Create user account in CRM system.
  4. Assign appropriate role-based permissions.
  5. Send credentials securely to the new user.

Account Termination Steps:

  1. Receive termination notification or HR exit form.
  2. Revoke system access and change passwords immediately.
  3. Archive user data for audit compliance.
  4. Deactivate account within 24 hours of departure.
  5. Update access logs and report completion.
